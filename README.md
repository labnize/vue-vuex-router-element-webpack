# Vue基础MVVM框架

## 技术及框架

包括但不限于:

- ES6/7  JS
- ESLint (代码规范)
- Vue (MVVM框架)
- Vuex (统一状态管理)
- Vue Router (路由)
- axios (http库)
- Element UI (UI框架)
- Babel (ES6/7代码转译浏览器可执行)
- Webpack (打包工具)

## 安装调试

```bash
npm i
# 或 yarn
npm run start
# 或 yarn start
```
## 打包

```bash
npm run build
# or
yarn build
```
